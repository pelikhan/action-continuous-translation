## Recursos

* Traduções alimentadas por IA para sua documentação.
* Integração perfeita com GitHub Actions.
* Atualizações incrementais de tradução usando inferência LLM.
* Utiliza Remark e manipulação de AST para traduções confiáveis.
* Controle de qualidade com LLM como juiz e verificações de compilador.
* Compatível com os formatos Markdown, GFM e MDX (até certo ponto).
* Integração com Astro Starlight
