## الميزات

* ترجمات مدعومة بالذكاء الاصطناعي لوثائقك.
* تكامل سلس مع GitHub Actions.
* تحديثات ترجمة تدريجية باستخدام استدلال LLM.
* يستخدم Remark ومعالجة AST لترجمات موثوقة.
* تحكم في الجودة باستخدام LLM كقاضٍ وفحوصات المترجم.
* يدعم تنسيقات Markdown وGFM وMDX (إلى حد ما).
* تكامل مع Astro Starlight

تم بناء هذا المشروع باستخدام [GenAIScript](https://microsoft.github.io/genaiscript).
