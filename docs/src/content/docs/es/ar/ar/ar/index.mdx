## Características

* Traducciones asistidas por inteligencia artificial para tu documentación.
* Integración fluida con GitHub Actions.
* Actualizaciones de traducción progresivas utilizando modelos LLM.
* Utiliza Remark y procesamiento AST para traducciones fiables.
* Control de calidad utilizando LLM como evaluador y verificaciones automáticas.
* Soporte para los formatos Markdown, GFM y MDX (parcialmente).
* Integración con Astro Starlight

Este proyecto está desarrollado utilizando [GenAIScript](https://microsoft.github.io/genaiscript).
