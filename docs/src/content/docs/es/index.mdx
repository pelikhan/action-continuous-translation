## Características

- Traducciones impulsadas por IA para tu documentación.
- Integración perfecta con GitHub Actions.
- Actualizaciones de traducción incremental utilizando inferencia LLM.
- Utiliza Remark y manipulación AST para traducciones fiables.
- Control de calidad con LLM-como-juez y verificaciones de compilador.
- Admite formatos Markdown, GFM y MDX (hasta cierto punto).
- Integración con Astro Starlight

Este proyecto está construido utilizando [GenAIScript](https://microsoft.github.io/genaiscript).
