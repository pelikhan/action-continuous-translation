## Recursos

* Traduções impulsionadas por IA para sua documentação.
* Integração perfeita com GitHub Actions.
* Atualizações de tradução incremental usando inferência de LLM.
* Usa Remark e manipulação de AST para traduções confiáveis.
* Controle de qualidade com LLM-como-juiz e verificações de compilador.
* Suporta formatos Markdown, GFM e MDX (até certo ponto).
* Integração com Astro Starlight
