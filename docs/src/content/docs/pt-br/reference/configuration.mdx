Os dados de configuração do frontmatter são definidos no arquivo `types.ts`.
Eles podem ser adicionados ao frontmatter de uma documentação como um objeto `translator`.

```markdown
---
translator:
  instructions: Use a friendly tone.
---

Lorem ipsum...
```
