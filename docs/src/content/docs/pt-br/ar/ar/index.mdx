## Recursos

* Traduções impulsionadas por IA para sua documentação.
* Integração perfeita com GitHub Actions.
* Atualizações de tradução incrementais usando inferência de LLM.
* Usa Remark e processamento AST para traduções confiáveis.
* Controle de qualidade com LLM como avaliador e verificações do tradutor.
* Suporte aos formatos Markdown, GFM e MDX (até certo ponto).
* Integração com Astro Starlight

Este projeto é construído usando o [GenAIScript](https://microsoft.github.io/genaiscript).
