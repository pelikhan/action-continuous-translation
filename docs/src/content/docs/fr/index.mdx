## Fonctionnalités

- Traductions alimentées par l'IA pour votre documentation.
- Intégration transparente avec GitHub Actions.
- Mises à jour incrémentales des traductions utilisant des inférences LLM.
- Utilise Remark et la manipulation AST pour des traductions fiables.
- Contrôle de qualité avec LLM en tant que juge et vérifications du compilateur.
- Prend en charge les formats Markdown, GFM et MDX (dans une certaine mesure).
- Intégration Astro Starlight

Ce projet est construit avec [GenAIScript](https://microsoft.github.io/genaiscript).
