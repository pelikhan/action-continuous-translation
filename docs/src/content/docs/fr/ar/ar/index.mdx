## Fonctionnalités

* Traductions assistées par intelligence artificielle pour vos documents.
* Intégration transparente avec GitHub Actions.
* Mises à jour progressives des traductions grâce à l'inférence LLM.
* Utilise Remark et le traitement AST pour des traductions fiables.
* Contrôle qualité avec LLM comme arbitre et vérifications des traducteurs.
* Prend en charge les formats Markdown, GFM et MDX (dans une certaine mesure).
* Intégration avec Astro Starlight

Ce projet est construit avec [GenAIScript](https://microsoft.github.io/genaiscript).
